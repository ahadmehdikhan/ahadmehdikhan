## Current Build: FANG

**FANG** is an ISO 27001 Gap Assessment Pipeline.

It is designed to take structured assessment data and convert it into validated findings, compliance scores, structured outputs, and professional reports.

Core capabilities:

* CSV intake for control assessment responses
* Input validation and normalization
* Deterministic compliance scoring
* Structured JSON output
* HTML and PDF report generation
* Optional AI-assisted narrative generation
* Consultant-style reporting for security assessments

FANG is not meant to replace professional judgment. It is meant to support it by making assessments more consistent, repeatable, and easier to deliver.

## How I Work

I approach security work with two questions:

**Does the organization have the right controls?**

**Can the system prove it clearly?**

A good security program should not depend on vague claims. It should produce evidence, show control status, identify gaps, and give clear next steps.

My goal is to build and assess security programs that are practical, defensible, and usable.
